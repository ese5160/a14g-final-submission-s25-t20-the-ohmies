SRS 01 – Water Level (FSR406)

Status: Achieved

  • ADC → pressureReading → volume via interpolation.
  • Sample every 5 s, accumulate intake.
uint16_t pressureReading = ADC_Read(Pressure_PIN);
float currentVolume = getVolumeFromAdc(pressureReading);
waterIntakeTotal += lastVolume - currentVolume;

SRS 02 – TDS on-demand ≤5 s

Status: Achieved

  • SEN0244 sensor, temp-compensation formula.
  • Measurement completes within 5 s.
float tdsValue = getTDS(temperature);
lcd_display("TDS Value", tdsValue);

SRS 03 – Heart Rate on-demand ≤10 s

Status: Achieved

  • MAX30102 PPG → BPM in ~12 s (<15 s req).
uint16_t pulseReading = ADC_Read(PULSE_PIN);
process_pulse(pulseReading);

SRS 04 – ESP32 → Blynk Platform

Status: Achieved

  • Wi-Fi streams BPM, intake, TDS to Blynk.
Blynk.virtualWrite(V0_BPM, bpm);
Blynk.virtualWrite(V1_WATER_INTAKE, waterintake);
Blynk.virtualWrite(V3_TDS, tdsValue);

SRS 05 – Hydration Reminders

Status: Achieved

  • LCD + Blynk reminder after simulated 25 s inactivity.
if (current_time - last_change_time >= 25000) {
  LCD_drawString(...);
  Blynk.virtualWrite(V2_DRINK, "Reminder: Drink Water");
}

SRS 06 – 1 s LCD Updates

LCD_drawString(...);  // heart rate
LCD_drawString(...);  // water level
LCD_drawString(...);  // TDS
