System Diagram

Team 20 "The Ohmies"

Signal Acquisition Unit

  • Water Intake Monitoring (FSR406): pressure readings → water volume.
  • Water Quality Testing (SEN0244): conductivity → TDS.
  • Pulse Acquisition (MAX30102): PPG → heart rate.

Local Monitoring Unit

  • ATmega328PB: processes intake, quality, & BPM.
  • LCD: real-time display.
  • Buzzer: alerts.
  • ESP32: Wi-Fi → cloud.

Remote Monitoring Unit

  • Cloud Platform:
    • Data Storage & Real-time Monitoring (Blynk).

Data Flow

  • Acquisition → Local processing & display → ESP32 → Cloud.
